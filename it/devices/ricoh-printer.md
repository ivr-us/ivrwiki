# Ricoh Printer advanced functions<Badge type="tip" text="Windows" />

:::tip
The printer admin credentials and Anwendercodes are stored i the [Password manager](../info/passwords)
:::

## Acessing the printer admin interface
Open the adress of the printer in the browser [141.58.162.8](http://141.58.162.8) and log in with the admin credentials.

## Changing the Anwendercode
1. Log in to the printer admin interface
2. Select `Device Management` and click on `Address book`
3. Select `Detail Input`, then the Kostenstelle X and click on `Change`
